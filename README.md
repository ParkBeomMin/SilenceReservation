# SilenceReservation

일정 시간동안 무음을 설정할 수 있습니다.
설정한 시간이 지나면, 소리모드로 전환됩니다.
위젯을 활용하여 30분, 1시간, 2시간, 4시간 단위로 무음을 쉽고 간편하게 설정할 수 있습니다.

더 이상 무음 설정을 해놓고 잊어 불편한 경험을 하지 마세요!

중요한 일이 있을 때, 방해 받고 싶지 않은 시간이 있을 때 무음 예약을 사용하세요!
